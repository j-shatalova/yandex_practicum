## Задачи проекта

- Исследовать таблицы — вывести первые строки;
- Сделать по одному SQL-запросу для решения каждого задания;
- Вывести результат каждого запроса в тетрадке;
- Описать выводы по каждой из решённых задач.

*sqlalchemy*
